Welcome to Gmail Priority AI

Thanks for installing the extension! The AI prioritization runs entirely on Chrome’s built-in Gemini Nano model, so the very first time you use it you’ll need to approve the download.

  1. Open gmail.com in a new tab.
  2. Click the Gmail Priority AI extension icon in your toolbar.
  3. Choose Enable AI Classification. This user gesture lets Chrome download the Gemini Nano model.
  4. Watch the inbox badges appear once AI reports “Built-in AI active”.

Pro tip: If you don’t see “Built-in AI active” after enabling, make sure you’re using Chrome Dev/Canary with the Prompt API flags turned on. You can revisit the popup any time to re-trigger the download.
